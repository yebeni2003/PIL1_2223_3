Rapport sur la base de données

Nom de la table Description
Etudiants Table pour stocker les informations des étudiants
Professeurs Table pour stocker les informations des Professeurs
Administrations Table pour stocker les informations sur des administrations
Filières Table pour stocker les informations sur des administrations
Salles Table pour stocker les informations sur les salles disponibles
UE Table pour stocker les informations sur les UE
Matières Table pour stocker les informations sur les matières des UE

Fonctionnalités implémentées

  • Ajout d'un nouvel étudiant avec ses informations de base (matricule_Etu, nom_Etu, prenom_Etu, sexe_Etu, date_nais_Etu, age_Etu, Contact, email_Etu);
  • Ajout d'un nouveau Professeurs avec ses informations (id_Prof, nom_Prof, prenom_Prof, sexe_Prof, contact_Prof, email_Prof );
  • Ajout d'un nouveau Administrations avec ses informations (id_Amin, nom_Admin, prenom_Admin, sexe_Admin, email_Admin);
  • Ajout d'un nouveau Filières avec ses informations (id_Fil, nom_Fil);
  • Ajout d'un nouveau Salles avec ses informations (id_Sal, nom_Sal, cap-Sal);
  • Ajout d'un nouveau UE avec ses informations (Code_UE, nom_UE, nombre_credit_UE, type_semestre_UE);
  • Ajout d'un nouveau Matières avec ses informations (id_Mat, nom_Mat, Horaire_Mat).

Améliorations possibles

  • Intégrer un système de connexion pour sécuriser l'accès à la base de données
  • Permettre la modification et la suppression des enregistrements existants
  • Ajouter des fonctionnalités de recherche et de filtrage pour faciliter la navigation dans l'emploi du temps
  • Générer des rapports statistiques sur la répartition des Matières et des étudiants
